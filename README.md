Mondy Cinema 🎬

A modern, database-driven **Movie Theater Booking System** built with PHP and MySQL. Designed for seamless movie browsing, real-time seat reservation, and efficient theater management.

---

## 🚀 Key Features

### 👤 Customer Experience
* **Secure Access:** Reliable user registration and authentication.
* **Movie Browsing:** Explore currently showing and upcoming movies.
* **Live Seat Selection:** Interactive visual map to select preferred seats.
* **Easy Booking:** Streamlined ticket reservation and checkout process.
* **Feedback:** Share experiences directly through the platform.

### ⚙️ Admin Control
* **Dashboard:** Real-time overview of bookings and system activity.
* **Content Management:** Effortlessly add, update, or remove movies and showtimes.
* **Booking Oversight:** Review and manage pending reservations.
* **User Management:** Maintain and oversee registered customer accounts.

---

## 🛠️ Technology Stack
* **Backend:** PHP (Native MySQLi with Prepared Statements for security).
* **Database:** MySQL / MariaDB.
* **Frontend:** HTML5, CSS3, JavaScript (AJAX/Fetch API).
